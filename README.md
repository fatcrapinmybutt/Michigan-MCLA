# Michigan-MCLA

A comprehensive catalogue of Michigan court abbreviations, administrative authority codes, and case type classification codes used throughout Michigan's legal system.

## Contents

- **[`catalogue.md`](catalogue.md)** — Human-readable catalogue with full descriptions, tables, and citation examples.
- **[`catalogue.json`](catalogue.json)** — Machine-readable JSON version of the complete catalogue.

## Coverage

The catalogue covers four main areas:

1. **Michigan State Court Abbreviations** — Mich. (MSC), Mich. App. (COA), Cir. Ct., Dist. Ct., Prob. Ct., Mun. Ct., JTC
2. **Federal Court Abbreviations** — U.S., 6th Cir., E.D. Mich., W.D. Mich.
3. **Court Rule & Administrative Authorities** — MCR, MRE, SCAO, MRPC, MCJC
4. **Case Type Classification Codes** — Circuit Court (AA, AR, AV, CH, DM, FY), District Court (GC, LT, SC, SM, SI), Probate Court (DA, DE, MI, JA)
