# cortex – CORTEX Intelligence Platform Legal Entity Detection Patterns

This directory contains the **CORTEX Intelligence Platform** legal entity
detection patterns, merged from the **cortex-legal** repository into this
unified Michigan-MCLA repository.

## Contents

| File | Description |
|---|---|
| `legal_starter.json` | Legal & Compliance Starter pack with entity patterns, categories, and authority definitions |

## Overview

The starter pack provides regex-based patterns for detecting legal entities such
as contract clauses, citations, regulations, courts, legal parties, actions,
monetary amounts, and Michigan-specific statutes. It also defines investigation
categories (Contract Analysis, Regulatory Compliance, Litigation, Michigan Family
Law, Michigan Criminal) and authority citation formats for US Code, CFR, MCL,
MCR, and MRE.
